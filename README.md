# Whatsapi

Ruby Interface to WhatsApp messenger

## Installation

Add this line to your application's Gemfile:

```ruby
	gem 'whatsapi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install whatsapi

## Usage

	### Create a client using the phone number and password

	```
		require 'whatsapi'

	```
